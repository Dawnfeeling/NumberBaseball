## 프로젝트 이름
숫자야구 게임
## 프로젝트 소개
숫자야구 게임을 구현

# 주요 기능
<게임 메뉴>

![img.png](img.png)

- 0을 입력하면 게임의 난이도를 설정한다. 이후 3, 4, 5 중에 원하는 난이도를 입력해서 설정한다.


- 1을 입력하면 게임을 시작한다. 설정한 난이도에 따라 랜덤으로 3, 4, 5 자리 숫자가 정답으로 생성된다.(중복 제외, 0 제외)
예측되는 숫자를 입력해서 정답을 맞추면 된다.
  - 숫자를 입력하면 정답과 비교해서 힌트를 출력한다.
  
    ![img_1.png](img_1.png)

  - 정답을 맞히면 게임 메뉴로 돌아온다


- 2를 입력하면 게임 기록을 불러온다.


- 3을 입력하면 게임을 종료한다.


- 게임을 종료하기 게임 메뉴 부분은 계속 반복한다.
